pyds
====


Introduction
~~~~~~~~~~~~

    This is the compilation of simple python pacakages, which is useful for data anaylsis in astrophysics. 
    All modules are compatible with both python 2.7 and 3.x. 
    (Module "future" is required to be pre-installed.)
    This is open source, which is permitted to be modified and spread. 
    The contact information is noted at https://astrodoo.github.io


Installation
~~~~~~~~~~~~

    Download files from 

    http://github.com/astrodoo/pyds

    or by git tools,

.. parsed-literal::

    git clone https://github.com/astrodoo/pyds.git


Dependence
~~~~~~~~~~
   
    This package depends on 

    * numpy
    * astropy
    * matplotlib
    * ffmpeg
